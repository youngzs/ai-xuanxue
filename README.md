# AI与玄学研究院 - Hexo博客

一个现代化、响应式的Hexo博客主题，专为技术博客和个人网站设计。

## 🚀 特性

### 🎨 现代化设计
- **响应式布局** - 完美适配桌面端、平板和移动设备
- **渐变色彩** - 使用现代渐变色彩方案
- **卡片式设计** - 清爽的卡片式文章展示
- **动画效果** - 流畅的CSS动画和过渡效果

### 📱 用户体验
- **移动端优化** - 汉堡菜单和触摸友好的界面
- **夜间模式** - 内置深色主题切换
- **回到顶部** - 平滑滚动回到页面顶部
- **阅读进度条** - 文章页面显示阅读进度

### 🛠 功能丰富
- **搜索功能** - 内置搜索框（可扩展）
- **社交链接** - 支持多种社交平台图标
- **代码高亮** - 支持代码块复制功能
- **文章统计** - 字数统计和阅读时间估算
- **分享功能** - 微博、QQ等社交分享

### 📊 侧边栏组件
- **个人资料** - 头像、简介和统计信息
- **最新文章** - 显示最近发布的文章
- **分类标签** - 文章分类和标签云
- **归档** - 按月份归档文章

### 🎨 AI工具集成
- **专业人像摄影提示生成器** - 集成FLUX模型的AI摄影工具
  - 📸 专业摄影参数设置
  - 🎨 多样化风格选择
  - ⚡ 即时生成高质量提示语
  - 🔄 支持批量创作
  - 🌙 深色模式支持
  - 📱 响应式设计

## 📁 项目结构

```
mysite/
├── _config.yml                 # Hexo主配置文件
├── themes/myblog/              # 自定义主题
│   ├── _config.yml            # 主题配置文件
│   ├── layout/                # 布局模板
│   │   ├── layout.ejs         # 主布局模板
│   │   ├── index.ejs          # 首页模板
│   │   ├── post.ejs           # 文章页模板
│   │   ├── page.ejs           # 页面模板
│   │   └── _partial/          # 组件模板
│   │       ├── header.ejs     # 头部导航
│   │       ├── footer.ejs     # 页脚
│   │       └── sidebar.ejs    # 侧边栏
│   └── source/                # 静态资源
│       ├── css/               # 样式文件
│       │   ├── style.css      # 主样式
│       │   ├── components.css # 组件样式
│       │   └── photography-tool.css # 摄影工具样式
│       ├── js/                # JavaScript文件
│       │   ├── main.js        # 主脚本
│       │   └── photography-tool.js # 摄影工具脚本
│       └── images/            # 图片资源
├── source/                     # 内容源文件
│   ├── _posts/                # 文章目录
│   ├── tools/                 # 工具页面
│   │   └── index.md          # AI工具集页面
│   └── about/                 # 关于页面
└── public/                    # 生成的静态网站
```

## 🚀 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
hexo server
```

### 3. 访问网站
打开浏览器访问 `http://localhost:4000`

### 4. 生成静态文件
```bash
hexo generate
```

## ⚙️ 配置说明

### 主题配置 (themes/myblog/_config.yml)

```yaml
# 导航菜单
menu:
  首页: /
  归档: /archives/
  分类: /categories/
  标签: /tags/
  工具: /tools/
  关于: /about/

# 社交链接
social:
  GitHub: https://github.com/your-username
  微博: https://weibo.com/your-username
  邮箱: mailto:your-email@example.com

# 侧边栏组件
sidebar:
  enable: true
  widgets:
    - profile
    - recent_posts
    - categories
    - tags
    - archive
```

## 🎨 自定义样式

### CSS架构
- `style.css` - 基础样式、布局、响应式设计
- `components.css` - 组件样式、文章样式、特效
- `photography-tool.css` - 摄影工具专用样式

### 颜色方案
- 主色调：`#3498db` (蓝色)
- 渐变色：`#667eea` → `#764ba2`
- 文字色：`#2c3e50` (深灰)
- 背景色：`#f8f9fa` (浅灰)

## 🎨 AI工具使用说明

### 专业人像摄影提示生成器

访问 `/tools/` 页面即可使用专业人像摄影提示生成器：

1. **基础设置**
   - 选择摄影风格（时尚、肖像、艺术等）
   - 设置场景环境（摄影棚、城市、自然等）
   - 配置姿势和表情
   - 选择构图和光线

2. **高级设置**
   - 艺术处理风格
   - 色彩方案调节
   - 镜头类型选择
   - 相机参数设置
   - 图像比例调整

3. **自定义功能**
   - 添加自定义关键词
   - 高分辨率选项
   - 质量提升词

4. **生成和管理**
   - 一键生成专业提示语
   - 复制到剪贴板
   - 保存历史记录
   - 随机选项功能

### 工具特色

- **专业级参数** - 涵盖专业摄影的各个方面
- **智能优化** - 自动添加质量提升关键词
- **批量支持** - 支持1-4张图片的批量生成设置
- **深色模式** - 自适应系统主题
- **响应式** - 完美适配各种设备
